JSON creation keyword that adds a string property given the {{name}} and {{value}} to the current JSON Object.

Example:

| `Start JSON Object`            |         |          |
| `Add JSON Object String Item`  | name    | config1  |
| `Add JSON Object Integer Item` | timeout | 1000     |
| `Add JSON Object Boolean Item` | enabled | true     |
| `End JSON Object`              |
| ${jsonString}=                 | `Create JSON String` |

{{jsonString}}:

| {
|    "name": "config1"
|    "timeout": 1000
|    "enabled": true
| }
